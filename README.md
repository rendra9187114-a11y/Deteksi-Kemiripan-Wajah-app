Projek Aljabar Linier

Menu 1: EDA & Evaluasi
Menyajikan analisis dataset wajah dan performa model PCA-SVD, meliputi jumlah data latih dan uji,
visualisasi mean face, explained variance, serta akurasi menggunakan Cosine Similarity dan Euclidean Distance.

Menu 2: Kompres Gambar
fitur untuk membandingkan gambar asli dengan hasil kompresi, sekaligus memberi kontrol penuh pada
user untuk memilih format dan kualitas, serta melihat statistik pengurangan ukuran file.

Menu 3: Deteksi Wajah dengan dataset
Membandingkan wajah pada gambar yang diunggah dengan seluruh wajah dalam dataset. Wajah direpresentasikan menggunakan PCA-SVD
dan dicocokkan menggunakan Cosine Similarity untuk menemukan identitas dengan tingkat kemiripan tertinggi.

Menu 4: Deteksi Wajah (Lama dan Sekarang)
Membandingkan dua foto dari orang yang sama pada waktu berbeda. Sistem mengekstraksi fitur wajah menggunakan PCA-SVD,
menghitung tingkat kemiripan dengan Cosine Similarity, dan menampilkan persentase kemiripan beserta interpretasinya.
